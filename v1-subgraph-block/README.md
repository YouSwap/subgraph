## 安装依赖
```shell
  yarn install
```

## 编译API文件
```shell
  yarn codegen
```

## 本地创建
```shell
  graph create block -g http://127.0.0.1:8020 -i http://127.0.0.1:5001 --debug
```

## 本地部署
```shell
  graph deploy block -g http://127.0.0.1:8020 -i http://127.0.0.1:5001 --debug
```
